## 🛠️ 准备工作

### 需要准备什么？

1. **一台电脑**（Windows、Mac、Linux都可以）
2. **现代浏览器**（推荐Chrome浏览器）
3. **稳定的网络连接**
4. **你的学号和密码**（用于登录学校选课系统）

### 检查浏览器版本

确保你的浏览器版本不要太旧：
- **Chrome**: 60版本以上
- **Firefox**: 55版本以上
- **Safari**: 10版本以上
- **Edge**: 79版本以上

**如何检查版本？**
1. 打开浏览器
2. 点击右上角的菜单按钮（通常是三个点或三条线）
3. 选择"设置" → "关于Chrome"（或类似选项）
4. 查看版本号

## 🎓 课程类型选择指南

### 新手推荐策略

如果你是第一次使用，建议按以下优先级选择：

1. **方案外选课 (KZYXK)** - ⭐⭐⭐⭐⭐
   - 最通用、成功率稳定
   - 适用于大部分选修课程

2. **推荐选课 (TJXK)** - ⭐⭐⭐⭐⭐
   - 如果有系统推荐的课程，优先选择

3. **方案内选课 (BFAK)** - ⭐⭐⭐⭐
   - 必修课程，配合推荐选课使用

### 如何在界面中选择课程类型

1. **添加课程时**：
   - 输入课程ID和名称
   - 在"课程类型"下拉框中选择合适的类型
   - 默认推荐使用"方案外选课"

2. **查看类型说明**：
   - 鼠标悬停在下拉框上会显示类型说明
   - 每种类型都有详细的适用场景描述

3. **实时验证**：
   - 系统会验证课程ID与类型的匹配性
   - 错误的类型组合会显示警告

### 课程类型选择示例

```javascript
// 示例：同时添加不同类型的课程
courseManager.addCourse('2024CS101', 'TJXK');  // 推荐选课 - 专业必修
courseManager.addCourse('2024ART301', 'KZYXK'); // 方案外选课 - 艺术选修
courseManager.addCourse('2024PE101', 'TYKXK');  // 体育选择课 - 体育选修
courseManager.addCourse('2024GEN201', 'QXGXK'); // 通识课程选修 - 志愿1
```

### 常见选择场景

**专业必修课** → 推荐选课 (TJXK) 或 方案内选课 (BFAK)
**通识选修课** → 方案外选课 (KZYXK) 或 通识课程选修 (QXGXK)
**体育课程** → 体育选择课 (TYKXK)
**重修课程** → 重修选课 (CXXK)
**创新创业课程** → 创新创业类选修课 (CXCY)

📖 **详细说明**：查看 [课程类型使用指南](course-types-guide.md) 获取更详细的信息。

## 📖 详细步骤说明

### 步骤1：登录学校选课系统

1. **打开浏览器**
   - 双击桌面上的浏览器图标（Chrome、Firefox等）

2. **输入网址**
   - 在地址栏输入：`https://xk.webvpn.scuec.edu.cn/xsxk/`
   - 按回车键

3. **登录系统**
   - 输入你的学号
   - 输入你的密码
   - 点击"登录"按钮

4. **确认登录成功**
   - 看到选课系统的主页
   - 确认右上角显示你的姓名或学号

### 步骤2：打开开发者控制台

**什么是开发者控制台？**
这是一个浏览器内置的工具，用来执行代码和查看网页信息。不用担心，我们只是用它来运行抢课助手。

**操作方法：**

1. **方法一：按F12键**
   - 按键盘上的 `F12` 键
   - 浏览器右侧或底部会出现开发者工具

2. **方法二：右键菜单**
   - 在网页任意位置右键点击
   - 选择"检查"或"检查元素"

3. **切换到Console标签**
   - 在开发者工具中找到"Console"标签
   - 点击切换到控制台界面

4. **清空控制台（可选）**
   - 点击控制台左上角的清除按钮🚫
   - 或在控制台输入 `clear()` 然后按回车

### 步骤3：准备和运行抢课助手

#### 方法A：单文件版本（推荐）

1. **获取代码**
   - 打开项目文件夹中的 `dist` 文件夹
   - 用文本编辑器打开 `course-helper.js` 文件
   - 按 `Ctrl+A` 全选代码
   - 按 `Ctrl+C` 复制

2. **粘贴并执行**
   - 回到浏览器控制台
   - 点击控制台输入区域
   - 按 `Ctrl+V` 粘贴代码
   - 按回车键执行

3. **确认运行成功**
   - 控制台显示："🎓 中南民族大学自动选课助手 V1.1.0 已启动"
   - 页面右下角出现"抢课"悬浮按钮
   - 界面底部显示作者信息和版本号

#### 方法B：油猴脚本版本

1. **安装Tampermonkey扩展**
   - Chrome：访问 Chrome Web Store 搜索 "Tampermonkey"
   - Firefox：访问 Firefox Add-ons 搜索 "Tampermonkey"
   - 安装并启用扩展

2. **创建脚本**
   - 点击浏览器工具栏中的油猴图标
   - 选择"管理面板"
   - 点击"➕"创建新脚本

3. **添加代码**
   - 清除默认模板代码
   - 复制 `dist/tampermonkey-course-helper.js` 的全部内容
   - 粘贴到编辑器中
   - 按 `Ctrl+S` 保存

4. **使用**
   - 访问选课系统
   - 脚本会自动运行
   - 直接开始使用即可

### 步骤4：使用抢课助手（V1.1.0）

#### 基础操作

1. **打开控制面板**
   - 点击右下角的"抢课"悬浮按钮
   - 展开完整的控制面板
   - 查看底部的作者信息和版本显示

2. **添加课程**
   - 在"输入课程ID"框中输入课程编号
   - 可选：在"输入课程名称"框中填入课程名称
   - 🆕 **选择课程类型**：在"课程类型"下拉框中选择合适的类型
   - 点击"添加更多课程"可添加多门课程

3. **开始选课**
   - 确认所有课程都已输入并选择了正确的类型
   - 点击"开始选课"按钮
   - 系统会自动开始监控并选课
   - 不同课程类型会使用对应的API端点

## 🔧 故障排除

### 数据持久化问题
**症状**: 页面刷新后课程信息丢失
**解决方法**:
1. 确认使用的是 V1.0.4 版本
2. 检查浏览器是否支持本地存储
3. 尝试重新添加课程，看是否能保存
4. 清除浏览器缓存后重试

### UI显示问题
**症状**: 控制面板显示异常或消失
**解决方法**:
1. 刷新页面重新开始
2. 检查是否有JavaScript错误
3. 尝试使用不同的浏览器
4. 确认代码复制完整

### 油猴脚本问题
**症状**: 油猴脚本不自动运行
**解决方法**:
1. 确认Tampermonkey扩展已启用
2. 检查脚本URL匹配规则
3. 查看脚本是否有语法错误
4. 重新安装脚本

### 性能问题
**症状**: 选课助手运行缓慢
**解决方法**:
1. 关闭其他不必要的标签页
2. 检查网络连接稳定性
3. 重启浏览器
4. 减少同时运行的课程数量
