## 🛠️ 准备工作

### 需要准备什么？

1. **一台电脑**（Windows、Mac、Linux都可以）
2. **现代浏览器**（推荐Chrome浏览器）
3. **稳定的网络连接**
4. **你的学号和密码**（用于登录学校选课系统）

### 检查浏览器版本

确保你的浏览器版本不要太旧：
- **Chrome**: 60版本以上
- **Firefox**: 55版本以上
- **Safari**: 10版本以上
- **Edge**: 79版本以上

**如何检查版本？**
1. 打开浏览器
2. 点击右上角的菜单按钮（通常是三个点或三条线）
3. 选择"设置" → "关于Chrome"（或类似选项）
4. 查看版本号

## 🚀 快速开始（3步搞定）

如果你对电脑比较熟悉，可以直接看这个简化版：

💡 **推荐使用方式**：
- **新手用户**：使用 `dist/course-helper.js` 单文件版本（一键运行）
- **油猴用户**：使用 `dist/tampermonkey-course-helper.js` 油猴脚本（自动运行）
- **开发者用户**：使用 `src/` 目录下的模块化版本（可定制）

## 🎯 三种使用方式详解

### 方式一：单文件版本 (⭐ 推荐新手)

**适合人群**：第一次使用的新手用户

**优点**：
- ✅ 一键运行，无需配置
- ✅ 包含所有功能，开箱即用
- ✅ 支持数据持久化（V1.0.4 新增）

**使用步骤**：
1. 下载项目文件
2. 打开 `dist/course-helper.js` 文件
3. 复制全部代码
4. 在选课系统控制台中粘贴并执行

### 方式二：油猴脚本版本 (⭐⭐ 最推荐)

**适合人群**：希望自动运行的日常用户

**优点**：
- ✅ 自动运行，无需手动操作
- ✅ 数据持久化，刷新页面不丢失
- ✅ 一次安装，永久使用

**安装步骤**：
1. 安装 Tampermonkey 扩展
2. 创建新脚本
3. 复制 `dist/tampermonkey-course-helper.js` 内容
4. 保存并启用
5. 访问选课系统即可自动运行

### 方式三：模块化版本 (⭐⭐⭐ 适合开发者)

**适合人群**：有一定技术基础的用户

**优点**：
- ✅ 高度可定制
- ✅ 模块化设计
- ✅ 便于二次开发

**使用步骤**：
```javascript
// 在控制台中依次执行
import { CONFIG } from './src/config.js';
import { courseManager } from './src/course-registration.js';
import { uiController } from './src/ui-controller.js';

// 初始化界面
uiController.initialize();
```

## 🆕 V1.0.4 新功能介绍

**这是什么？**
- 课程信息会自动保存到浏览器本地

- 显示所有课程管理功能

- 选课进行中自动切换到此状态
- 实时显示选课进度和成功数量
- 节省屏幕空间，不影响其他操作

### 🎯 智能滚动容器

**自动启用条件**：
- 当添加超过4门课程时
- 自动出现滚动条
- 保持界面整洁美观

### 📊 增强的状态监控

**新增功能**：
- 运行时间统计（精确到秒）
- 详细的状态弹窗
- 实时进度显示
- 实验班数量显示

## 📖 详细步骤说明

### 步骤1：登录学校选课系统

1. **打开浏览器**
   - 双击桌面上的浏览器图标（Chrome、Firefox等）

2. **输入网址**
   - 在地址栏输入：`https://xk.webvpn.scuec.edu.cn/xsxk/`
   - 按回车键

3. **登录系统**
   - 输入你的学号
   - 输入你的密码
   - 点击"登录"按钮

4. **确认登录成功**
   - 看到选课系统的主页
   - 确认右上角显示你的姓名或学号

### 步骤2：打开开发者控制台

**什么是开发者控制台？**
这是一个浏览器内置的工具，用来执行代码和查看网页信息。不用担心，我们只是用它来运行抢课助手。

**操作方法：**

1. **方法一：按F12键**
   - 按键盘上的 `F12` 键
   - 浏览器右侧或底部会出现开发者工具

2. **方法二：右键菜单**
   - 在网页任意位置右键点击
   - 选择"检查"或"检查元素"

3. **切换到Console标签**
   - 在开发者工具中找到"Console"标签
   - 点击切换到控制台界面

4. **清空控制台（可选）**
   - 点击控制台左上角的清除按钮🚫
   - 或在控制台输入 `clear()` 然后按回车

### 步骤3：准备和运行抢课助手

#### 方法A：单文件版本（推荐）

1. **获取代码**
   - 打开项目文件夹中的 `dist` 文件夹
   - 用文本编辑器打开 `course-helper.js` 文件
   - 按 `Ctrl+A` 全选代码
   - 按 `Ctrl+C` 复制

2. **粘贴并执行**
   - 回到浏览器控制台
   - 点击控制台输入区域
   - 按 `Ctrl+V` 粘贴代码
   - 按回车键执行

3. **确认运行成功**
   - 控制台显示："中南民族大学自动选课助手 v1.0.4 已启动"
   - 页面右下角出现"抢课"悬浮按钮

#### 方法B：油猴脚本版本

1. **安装Tampermonkey扩展**
   - Chrome：访问 Chrome Web Store 搜索 "Tampermonkey"
   - Firefox：访问 Firefox Add-ons 搜索 "Tampermonkey"
   - 安装并启用扩展

2. **创建脚本**
   - 点击浏览器工具栏中的油猴图标
   - 选择"管理面板"
   - 点击"➕"创建新脚本

3. **添加代码**
   - 清除默认模板代码
   - 复制 `dist/tampermonkey-course-helper.js` 的全部内容
   - 粘贴到编辑器中
   - 按 `Ctrl+S` 保存

4. **使用**
   - 访问选课系统
   - 脚本会自动运行
   - 直接开始使用即可

### 步骤4：使用抢课助手（V1.0.4）

#### 基础操作

1. **打开控制面板**
   - 点击右下角的"抢课"悬浮按钮
   - 展开完整的控制面板

2. **添加课程**
   - 在"输入课程ID"框中输入课程编号
   - 可选：在"输入课程名称"框中填入课程名称
   - 点击"添加更多课程"可添加多门课程

3. **开始选课**
   - 确认所有课程都已输入
   - 点击"开始选课"按钮
   - 系统会自动开始监控并选课

#### V1.0.4 新增功能使用

**💾 数据持久化体验**：
- 输入课程后，即使刷新页面，课程信息会自动恢复
- 课程名称也会被保存
- 选课状态（成功/失败）会被记录

**🎨 UI状态切换**：
- 开始选课后，面板会自动最小化
- 点击最小化面板可以查看实时状态
- 可以再次点击展开完整面板进行管理

**📊 查看详细状态**：
- 点击"查看状态"按钮
- 打开可拖拽的状态详情弹窗
- 显示总课程数、成功数量、运行时间等详细信息

**🔄 动态管理课程**：
- 选课进行中可以随时添加新课程
- 可以删除不需要的课程
- 可以修改课程ID或名称

### 步骤5：监控和管理

#### 实时监控

1. **控制台日志**
   - 打开控制台查看详细日志
   - 成功抢到的课程会显示 ✅ 标记
   - 失败的尝试会显示具体原因

2. **迷你状态面板**
   - 选课进行中，控制面板会自动最小化
   - 实时显示：成功数量/总数量、运行时间
   - 可以直接在迷你面板中停止选课

3. **状态详情弹窗**
   - 点击"查看状态"查看详细信息
   - 包含每个课程的具体状态
   - 显示实验班信息和就绪状态

#### 高级功能

**⚠️ 智能确认机制**：
- 删除课程时会出现确认对话框
- 重置系统前会显示详细风险提示
- 关闭程序时有分级警告系统

**🎯 智能滚动**：
- 添加超过4门课程时自动启用滚动
- 自定义滚动条样式，保持美观

**🛑 自动停止**：
- 课程列表为空时自动停止选课
- 避免无效的API请求

### 步骤6：抢课成功后的操作

1. **确认抢课成功**
   - 控制台显示"🎉 选课成功! 课程: [课程ID]"
   - 状态面板中课程显示为"✅ 已成功"
   - 可以在选课系统中确认已选课程

2. **停止抢课助手**
   - 如果还有其他课程需要抢，可以继续运行
   - 所有课程都成功后，点击"停止选课"按钮

3. **关闭程序**（可选）
   - 点击控制面板右上角的 ✕ 按钮
   - 确认关闭，程序会自动清理所有资源

4. **备份选课信息**
   - 截图保存已选课程列表
   - 记录选课时间和课程信息

## 🔧 故障排除

### 数据持久化问题
**症状**: 页面刷新后课程信息丢失
**解决方法**:
1. 确认使用的是 V1.0.4 版本
2. 检查浏览器是否支持本地存储
3. 尝试重新添加课程，看是否能保存
4. 清除浏览器缓存后重试

### UI显示问题
**症状**: 控制面板显示异常或消失
**解决方法**:
1. 刷新页面重新开始
2. 检查是否有JavaScript错误
3. 尝试使用不同的浏览器
4. 确认代码复制完整

### 油猴脚本问题
**症状**: 油猴脚本不自动运行
**解决方法**:
1. 确认Tampermonkey扩展已启用
2. 检查脚本URL匹配规则
3. 查看脚本是否有语法错误
4. 重新安装脚本

### 性能问题
**症状**: 选课助手运行缓慢
**解决方法**:
1. 关闭其他不必要的标签页
2. 检查网络连接稳定性
3. 重启浏览器
4. 减少同时运行的课程数量
